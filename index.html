Welcome to the Podcast Generator

Here you will find details of the Podcast Generator that has been developed to allow a quick, automated way to take a folder structure and turn it into a podcast. Combined with Dropbox, you can have a podcast live in seconds. To get a podcast up and running, just alter the supported-files.xml to allow your file format, then create a configuration file. Set up your classpath like this (this is for Windows - you Linux guys know what your doing!)


echo off
setlocal enabledelayedexpansion
set cp=lib;config;.;./podcastGenerator.jar
for %%f in (lib/*.jar) do (set cp=!cp!;lib/%%f%)

Then run it like so...


$ java -cp %cp% com.morty.podcast.writer.PodCastCreationRunner %1 > %2

Configurable Filenames

The Podcast Generator allows different filename formats, and can be customised to reject certain files, allowing you to store files in the podcast folders that you dont want published. On top of this custom descriptions and filename fields allow for unique descriptions on each of your files on the podcast.

Automated

And yes - its easy to automate. Just throw it in the crontab (Linux, Mac OSX) or the Task Schedular (Windows) and you have a fully up-to-date syncing mechanism. Combine your podcast folder with Dropbox and you will never have to even think about syncing things again! You save the file, Dropbox uploads, and the Podcast Generator looks after the publishing.
